Synthwave '84 Code Blocks

This page demonstrates custom code blocks with syntax highlighting powered by Prism.js. To learn how to use these code blocks on your own page, see the Instructions and Usage Notes. There is also a brief guide on How to Escape HTML Characters in <code> tags at the bottom of this page.

Features

  • Synthwave '84 theme for code blocks with vibrant syntax highlighting, modified slightly from the original to include a custom linear-gradient background.
  • Language-specific syntax highlighting for: HTML, CSS, JavaScript, Bash, Python
  • Copy button to easily copy code snippets. Copy button updated to work on SoloLearn!
  • Custom font: Share Tech Mono.
  • Optional line numbers: See the HTML code block below for a demo.

Code Block Instructions

Prism.js

The Core Prism.js library includes syntax highlighting for HTML, CSS, and JavaScript. The core library is required. You can include other languages by linking to their Prism.js libraries. Add the following to index.html above the closing </body> tag to include syntax highlighting for HTML, CSS, JavaScript, Bash, and Python: 

<!-- Required - Prism.js Core (HTML, CSS, JavaScript) -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/prism/1.29.0/prism.min.js"></script>
<!-- Optional - Bash -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/prism/1.29.0/components/prism-bash.min.js"></script>
<!-- Optional - Python -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/prism/1.29.0/components/prism-python.min.js"></script>

Synthwave '84 Theme

Add the Synthwave '84 syntax highlighting theme to your code blocks by including the following in the <head> of index.html: 

<!-- Synthwave '84 Syntax Highlighting -->
<link rel="stylesheet" href="https://gen-w-deja.github.io/prism-themes/themes/prism-synthwave84.css">

Copy Button

Add this to script.js: 

// JavaScript function to copy code to clipboard with fallback for older browsers
// (or playgrounds like SoloLearn)
function copyCode(button) {
  const code = button.nextElementSibling.textContent;

  // Modern method: navigator.clipboard
  if (navigator.clipboard) {
    navigator.clipboard.writeText(code)
      .then(() => {
        button.textContent = 'Copied!';
        setTimeout(() => {
          button.textContent = 'Copy';
        }, 2000);
      })
      .catch(err => {
        console.error('Failed to copy text: ', err);
      });
  } else {
    // Fallback method: textarea and execCommand
    const tempInput = document.createElement('textarea');
    tempInput.value = code;
    document.body.appendChild(tempInput);
    tempInput.select();

    try {
      document.execCommand('copy');
      button.textContent = 'Copied!';
      setTimeout(() => {
        button.textContent = 'Copy';
      }, 2000);
    } catch (err) {
      console.error('Failed to copy text: ', err);
    }

    document.body.removeChild(tempInput);
  }
}

Line Numbers

Line numbers can optionally be included by adding this in the <head> of index.html: 

<!-- Optional - Prism.js line numbers - Stylesheet -->
<link href="https://cdnjs.cloudflare.com/ajax/libs/prism/1.29.0/plugins/line-numbers/prism-line-numbers.min.css" rel="stylesheet" />

And add this to index.html above the closing </body> tag: 

<!-- Optional - Prism.js line numbers - Script -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/prism/1.29.0/plugins/line-numbers/prism-line-numbers.min.js"></script>

Custom Background

To use the custom linear-gradient background featured in these code blocks, include this in styles.css: 

:not(pre) > code[class*="language-"],
pre[class*="language-"] {
  /* Code block background gradient */
  background-image: linear-gradient(to bottom, #2a2139, #34294f, #8f298f80) !important;
}

pre {
  /* Prevents discolored gaps at the top and bottom of code blocks */
  margin: 0 !important;
}

Custom Font

To change the code block font to Share Tech Mono, add this to the <head> of index.html: 

<!-- Share Tech Mono Font for Code Blocks -->
<link href="https://fonts.googleapis.com/css2?family=Share+Tech+Mono&display=swap" rel="stylesheet">

Use class="code-block" in the html <div>(s) containing your code, and add this to styles.css: 

.code-block code {
  font-family: 'Share Tech Mono', monospace !important;
}

For the remaining custom code block styling used here, use class="code-block" in the html <div>(s) containing your code, and add this to styles.css:

Usage Notes

To insert a code block in index.html, follow this structure: 

<!-- Opening tags -->
<div class="code-block">
<button class="copy-btn" onclick="copyCode(this)">Copy</button>
<!-- Line numbers are optional.
Remove the line-numbers class from pre to exclude them.
(but keep the pre tag!) -->
<pre class="line-numbers">
<!-- Insert language in the code class, after the hyphen.
This example uses html. -->
<code class="language-html">

<!-- Insert code here -->

<!-- Closing tags -->
</code></pre>
</div>

Here is the same code as above, without comments, for easier copy-paste, using 'CODE' as placeholder text: 

<div class="code-block"><button class="copy-btn" onclick="copyCode(this)">Copy</button>
    <pre class="line-numbers"><code class="language-html">CODE</code></pre>
</div>

The above code would display this: 

CODE


How to Escape HTML Characters

Certain HTML characters within <code> tags must be escaped to prevent the browser from attempting to interpret them.

Unescaped Character Escaped Representation
< &lt;
> &gt;
& &amp;
" &quot;
' &apos;

Example

Unescaped HTML: 

<div class="example">
    <p>This is a paragraph.</p>
</div>

To include the above HTML within <code> tags, the Escaped HTML would look like this: 

&lt;div class=&quot;example&quot;&gt;
    &lt;p&gt;This is a paragraph.&lt;/p&gt;
&lt;/div&gt;
